# 🏠 Y-Plaza

## 📌 Description
Y-Plaza est une application simple de gestion de biens immobiliers.
Le projet met en place une architecture client-serveur avec un backend et un frontend.

## ⚙️ Technologies
- Node.js
- Express
- HTML
- Git / GitHub

## 🚀 Fonctionnement

### Backend
Une API avec une route `/biens` qui retourne des données au format JSON.

### Frontend
Une page HTML qui utilise fetch pour récupérer les données et les afficher.

## ▶️ Lancer le projet

npm install
node server.js

Puis ouvrir le fichier `index.html` dans un navigateur et cliquer sur "Afficher les biens".
